# Librería de Componentes

Esta carpeta contiene los componentes UI reutilizables unificados para el proyecto.

## Archivos incluidos
- `Boton.html`: variantes de botones primarios, secundarios, outlines, iconos y botones wireframe.
- `Sidebar.html`: barra lateral reutilizable con estructura de logo, perfil, navegación y cierre de sesión.
- `Table.html`: tabla estándar con bordes y estilos comunes del proyecto.
- `FormField.html`: campos de formulario reutilizables (input, select, radio, checkbox).
- `Dropzone.html`: zona de carga con estilo drag & drop para archivos.
- `Card.html`: tarjetas de KPI y módulos con bordes y etiquetas.
- `Badge.html`: badges de estado y categoría.

## Propósito
Los archivos de esta carpeta reemplazan las versiones duplicadas de componentes que existían en múltiples páginas.

## Uso
Copiar y adaptar el HTML de estos componentes en las páginas según sea necesario, manteniendo los estilos compartidos.
